초등돌봄·교육 수요조사를 위한 가장 스마트한 솔루션
설치부터 관리까지, 누구나 쉽게 시작할 수 있습니다.

시작하기 학교 맞춤 설정 자주 묻는 질문

시스템의 두 가지 핵심 강점

학부모님께는 편리함을, 선생님께는 업무 효율을 선물합니다.

디자인 (직관성)

직관적인 선택 및 시간표 그리드

학부모는 색상으로 구분된(핑크, 노랑, 초록 등) 시간표에서 원하는 프로그램을 한눈에 확인하고 선택합니다.

데이터 (자동화)

실시간 집계 및 구조화

신청 내역이 Google Sheets에 실시간으로 저장되며, 귀가 정보와 동행자 정보가 별도 열에 체계적으로 정리되어 데이터 관리가 매우 용이합니다.

설치 및 배포 가이드

Google 계정만 있다면 5분 안에 시스템을 구축할 수 있습니다.

1

교육청 계정 로그인

먼저 브라우저에서 교육청 워크스페이스 계정(@dge.go.kr)으로 로그인해주세요.

2

사본 만들기

아래 링크를 클릭하여 구글 스프레드시트 사본을 생성합니다.

[클릭] 톡톡 신청서 사본 만들기
3

스크립트 편집기 열기

생성된 스프레드시트 상단 메뉴에서 확장 프로그램 > Apps Script를 클릭하여 스크립트 편집기를 엽니다.

4

설정 수정

좌측 파일 목록에서 settings.gs를 클릭하고, 학교 상황에 맞게 옵션을 수정합니다.

5

웹 앱으로 배포

우측 상단 배포 > 새 배포를 클릭하고 아래와 같이 설정합니다.

  • 유형 선택: 웹 앱
  • 설명: (자유롭게 입력)
  • 실행 계정: 나 (웹 앱을 실행하는 사용자)
  • 액세스 권한: 모든 사용자 (학부모님이 로그인 없이 접속 가능하도록)
중요: '액세스 권한'을 반드시 '모든 사용자'로 설정해야 학부모님이 구글 로그인 없이 설문에 참여할 수 있습니다.
6

액세스 승인 (최초 1회)

처음 배포 시 권한 승인 팝업이 뜹니다. 아래 순서대로 진행해주세요.

  1. 액세스 승인 클릭
  2. 사용할 구글 계정 선택
  3. '액세스 승인' 창이 뜨면 모든 항목에 체크(v)한 후 Continue(계속)를 클릭하여 승인합니다.
7

URL 공유

배포가 완료되면 생성된 웹 앱 URL을 복사하여 가정통신문이나 문자로 학부모님께 안내합니다.

학교 맞춤 설정

settings.gs 파일을 수정하여 우리 학교에 딱 맞는 시스템을 만드세요.
형광펜으로 표시된 부분만 수정하시면 됩니다.

1. 기본 정보 및 개인정보 수집 동의

학교 이름, 연락처, 그리고 개인정보 수집 동의 문구를 설정합니다.

팁: 여러 줄의 긴 문장을 입력할 때는 백틱(`) 기호로 문장을 감싸야 합니다. (작은따옴표 ' 아님) 
basic: {
  schoolName: '동도초등학교',
  schoolContact: '053-123-5678',
  title: '2025학년도 초등돌봄·교육 수요조사',
  subtitle: '희망하는 프로그램을 선택해주세요.',
  
  // 개인정보 수집 안내 (백틱 ` 사용)
  privacyNotice: `본교는 교육활동을 위해...
  (줄바꿈 가능)
  ...동의합니다.`
}

2. 알림사항 (상세 설명) 설정

설문 상단에 표시될 공지사항이나 안내문을 설정합니다.

description: {
  enabled: true,  // 표시 여부 (true: 표시, false: 숨김)
  title: '알림사항',
  
  // 안내 문구 (백틱 ` 사용)
  content: `주요 특징:
  • 맞춤형 프로그램: ...
  • 돌봄: ...`
}

3. 기능 활성화 설정

우리 학교에서 운영하는 프로그램만 켜고 끌 수 있습니다. (true: 사용, false: 미사용)

features: {
  enableMorningCare: true,   // 아침돌봄
  enableEveningCare: true,   // 저녁돌봄
  enableDismissalTime: false, // 하교시간 입력 (돌봄 학생 대상)
  showPeriodTimes: true     // 시간표에 시간 표시 여부
}

4. 교시 및 시간 설정

운영하는 교시를 정의합니다. 요일별로 시간이 다르다면 텍스트로 자유롭게 입력하세요.

periods: [
  // 단순 시간 표시
  { label: '1교시', time: '13:00-13:40' },
  
  // 요일별 시간 표시 예시
  { label: '2교시', time: '월금 13:50-14:30, 화수목 14:00-14:40' },
  
  // 시간 미표시 (공란)
  { label: '3교시', time: '' },
]

5. 과목(프로그램) 상세 설정

각 과목별로 운영 방식을 세밀하게 조정할 수 있습니다.

subjects: [
  {
    name: '맞춤형', // 시스템 고정 이름
    
    // 1. 사용 여부
    enabled: true, 
    
    // 2. 선택 가능한 교시 (이 과목이 개설된 교시만 입력)
    // 예: 1, 2교시만 운영하면 [1, 2]
    availablePeriods: [1, 2],
    
    // 3. 시간 묶음 (블록 타임)
    // true면 아래 지정된 교시들이 한 번에 선택됩니다. (예: 2시간 연속 수업)
    useTimeBundle: true,
    timeBundlePeriods: [1, 2],
    
    // 4. 요일 묶음 (주 5일제 등)
    // true면 하루를 선택해도 해당 요일들이 모두 같이 선택됩니다.
    useDayBundle: false,
    dayBundleDays: []
  },
  // ...
]

실무사님의 효율적인 업무 활용

클릭 한 번으로 끝내는 관리! 수요조사 마감 후, 구글 스프레드시트의 '톡톡 관리' 메뉴를 활용하세요.

업무 구분 구체적인 활용 방법 (관리자 메뉴) 업무 경감 효과
통계 및 현황 보고 '톡톡 관리 > 통계 페이지 생성' 클릭 전체 응답 현황이 시각적인 그래프로 자동 생성되어, 미달/초과 현황을 즉시 파악하고 보고합니다.
프로그램별 명단 확보 '톡톡 관리 > [프로그램명] 페이지 생성' 클릭 해당 프로그램 신청 학생 명단만 따로 정리되어 추출되므로, 수기 취합 없이 바로 출석부로 활용할 수 있습니다.
요일별 귀가 관리 수합 시트에서 하교 방법/동행자 연락처 열 필터링 보호자 동행 필요 학생의 당일 명단을 쉽게 확보하여 안전 관리의 정확성 및 긴급 연락 시스템을 강화합니다.

시스템 활용 핵심

톡톡 수강신청폼은 신청의 직관성을 높여 학부모의 부담을 줄이고,
수집된 데이터를 자동 관리 및 추출함으로써 업무를 최소화하는 데 가장 큰 도움이 됩니다.

자주 묻는 질문

Q. 수정을 했는데 반영이 안 돼요.

A. Apps Script 코드를 수정했다면 반드시 '새 배포'를 해야 사용자에게 반영됩니다. '배포 관리'가 아닌 '새 배포'를 눌러 버전을 업데이트해주세요.

Q. 사본 만들기 시 '액세스 권한 필요'라고 떠요.

A. 교육청 계정이 아닌 개인 계정으로 접속해서 그렇습니다. 화면 하단의 계정명을 클릭하여 교육청 계정으로 전환해주세요.

Q. 교육청 워크스페이스 계정은 어떻게 만드나요?

A. 학교 정보부(정보 담당 선생님)에 문의하시면 발급받으실 수 있습니다.

Q. "권한이 필요합니다"라고 떠요.

A. 배포 시 '액세스 권한'을 '모든 사용자'로 설정하지 않아서 그렇습니다. 다시 배포하여 권한 설정을 확인해주세요.

Q. 설문을 마감(종료)하고 싶어요.

A. 우측 상단 배포 > 배포 관리 메뉴로 이동하여, 현재 활성화된 배포 버전을 '보관(Archive)' 처리하면 더 이상 접속할 수 없게 됩니다.

Q. 신청 내역을 삭제하고 싶어요.

A. '응답데이터' 시트에서 해당 행을 삭제하면 됩니다. 통계는 다시 '통계 페이지 생성'을 실행하면 갱신됩니다.
